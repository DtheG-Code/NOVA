Willkommen in NOVA Studio

Eine vollständig abgeschottete Windows-/PC-VM, die komplett in NOVA läuft. Sie lebt im WebAssembly-Sandkasten — selbst Schadsoftware darin kann niemals auf deinen echten Rechner übergreifen.
